# Image Prompts for M M Detalhe Automóvel

Generated for AI image tools (Gemini, Midjourney, DALL-E, Flux, etc.)

## Tips
- Generate all images in one category before moving to the next for visual consistency
- Use the same seed/style reference across a category if your tool supports it
- Dark, moody, premium automotive aesthetic throughout — dramatic lighting, deep shadows
- Cars should be European models (BMW, Mercedes, Audi, VW) to match Portuguese market
- JPEG or WebP at 80% quality
- Save images to the project's `public/` directory matching the exact paths listed
